# Foundations of Machine Learning – Lab Assignment 8

This repository contains the solution for Lab Assignment 8 of the Foundations of Machine Learning practical course.

The assignment focuses on:
- Decision Tree classification
- Ensemble learning
- Random Forest concepts
- Gini impurity
- Decision boundaries
- Classification evaluation metrics

using Python and machine learning libraries.

---

## Topics Covered

### 1. Decision Tree Classification
- Training Decision Tree classifiers using scikit-learn
- Splitting datasets into train and test subsets
- Predicting class labels
- Evaluating model performance using F1-Score
- Visualizing learned decision boundaries

### 2. Ensemble Methods – Random Forest Concepts
- Bootstrapping datasets
- Training multiple Decision Trees
- Majority voting for ensemble prediction
- Computing Cross-Entropy loss
- Visualizing ensemble prediction probabilities

### 3. Gini Impurity
Implementation of:
- Gini impurity calculation
- Dataset splitting based on feature thresholds
- Weighted split impurity computation
- Visualization of split positions and impurity values
