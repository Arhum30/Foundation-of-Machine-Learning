# Foundations of Machine Learning – Lab Assignment 7

This repository contains the solution for Lab Assignment 7 of the Foundations of Machine Learning practical course.

The assignment focuses on:
- Principal Component Analysis (PCA)
- One-vs-Rest multiclass classification
- Softmax classification
- Cross-entropy loss
- PyTorch-based machine learning models
- Dimensionality reduction and visualization

using Python, NumPy, PyTorch, and scikit-learn.

---

## Topics Covered

### 1. Linear Classifier for Multiple Classes
- Loading and preprocessing the MNIST digits dataset
- Splitting datasets into train and test subsets
- Implementing PCA for dimensionality reduction
- Implementing One-vs-Rest multiclass classification
- Evaluating classification accuracy
- Visualizing predicted and ground truth labels in 2D PCA space

### 2. Softmax Classification
- One-hot encoding class labels
- Implementing the softmax activation function
- Building a Softmax classifier using PyTorch
- Implementing cross-entropy loss
- Training models using Stochastic Gradient Descent (SGD)
- Plotting loss curves during training
- Evaluating model performance on test data
