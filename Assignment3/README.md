# Foundations of Machine Learning – Lab Assignment 3

This repository contains the solution for Lab Assignment 3 of the Foundations of Machine Learning practical course.

The assignment focuses on:
- Dimensionality reduction
- Principal Component Analysis (PCA)
- t-SNE fundamentals
- Eigenvalue decomposition
- Probability-based neighborhood modeling

using Python and scientific computing libraries.

---

## Topics Covered

### 1. Dimensionality Reduction
- Understanding the curse of dimensionality
- PCA and t-SNE concepts
- Advantages and limitations of dimensionality reduction methods

### 2. Principal Component Analysis (PCA)
- Loading and visualizing the MNIST digits dataset
- Computing feature means and centering data
- Computing covariance matrices manually
- Eigenvalue and eigenvector decomposition
- Selecting principal components
- Projecting high-dimensional data into lower dimensions
- Visualizing projected parameter spaces

### 3. t-SNE Fundamentals
Implementation of:
- Conditional probabilities
- Perplexity computation
- Binary search optimization for sigma values
- Neighborhood probability distributions
