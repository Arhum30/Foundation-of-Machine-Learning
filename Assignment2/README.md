# Foundations of Machine Learning – Lab Assignment 2

This repository contains the solution for Lab Assignment 2 of the Foundations of Machine Learning practical course.

The assignment focuses on:
- Information theory
- Entropy analysis
- Statistical distance measures
- Probability distributions
- Mahalanobis distance
- KL divergence using Python

---

## Topics Covered

### 1. Shannon Entropy
- Computing probabilities of labels
- Implementing entropy calculations
- Comparing binary and multi-class entropy
- Understanding uncertainty in datasets

### 2. Mahalanobis Distance
- Computing covariance matrices
- Implementing Mahalanobis distance
- Comparing Mahalanobis and Euclidean distance
- Pearson correlation analysis

### 3. Kullback–Leibler (KL) Divergence
Implementation of:
- KL divergence for discrete distributions
- KL divergence interpretation
- Comparison of probability distributions
- Analytic KL divergence for Gaussian distributions
