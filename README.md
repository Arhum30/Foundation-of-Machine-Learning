# Foundations of Machine Learning – Lab Assignment 1

This repository contains the solution for Lab Assignment 1 of the Foundations of Machine Learning practical course.

The assignment focuses on:
- Data preprocessing
- Visualization methods
- Correlation analysis
- Similarity metrics using Python

---

## Topics Covered

### 1. Handling Tabular Data
- Creating and manipulating datasets using pandas
- Adding new patient records
- Feature normalization using Min-Max Scaling
- Saving datasets as CSV files
- Filtering data based on statistics

### 2. Data Visualization
- Histograms using matplotlib
- Scatter plots for correlation analysis
- Visualization of normalized features

### 3. Statistical Analysis
- Pearson Correlation Coefficients
- Feature relationship interpretation

### 4. Similarity Metrics
Implementation of:
- L1 Norm (Manhattan Distance)
- L2 Norm (Euclidean Distance)
- Cosine Similarity

- NumPy
