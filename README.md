# Foundations of Machine Learning

This repository contains practical lab assignments and implementations completed for the Foundations of Machine Learning course at TH Ingolstadt.

## Topics Covered

- Data preprocessing
- Statistical analysis
- Similarity metrics
- PCA and t-SNE
- k-Nearest Neighbors (k-NN)
- Linear Classification
- Softmax Classification
- Decision Trees
- Random Forests
- Neural Networks
- Model Evaluation    
- ROC Curves
- Ensemble Learning
