# Foundations of Machine Learning – Lab Assignment 6

This repository contains the solution for Lab Assignment 6 of the Foundations of Machine Learning practical course.

The assignment focuses on:
- k-Nearest Neighbors (k-NN) classification
- Principal Component Analysis (PCA)
- Hyperparameter tuning
- Linear classification
- Least Squares classification
- Dimensionality reduction

using Python and machine learning libraries.

---

## Topics Covered

### 1. k-Nearest Neighbors (k-NN) Classification
- Splitting datasets into train, test, and validation subsets
- Implementing PCA from scratch
- Dimensionality reduction to lower feature spaces
- Implementing a custom k-NN classifier
- Hyperparameter optimization for different k-values
- Accuracy evaluation
- Visualization of predicted and ground truth labels

### 2. Linear Classification
- Binary classification using Least Squares
- Adding bias terms to feature matrices
- Computing linear classifier weights
- Predicting class labels
- Visualizing decision boundaries
- Comparing predictions with ground truth labels
