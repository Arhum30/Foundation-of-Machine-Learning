# Foundations of Machine Learning – Lab Assignment 4

This repository contains the solution for Lab Assignment 4 of the Foundations of Machine Learning practical course.

The assignment focuses on:
- Overfitting and underfitting
- Polynomial regression
- Model evaluation
- ROC curves
- Classification metrics
- Performance comparison of predictive models

using Python and machine learning libraries.

---

## Topics Covered

### 1. Overfitting and Underfitting
- Generating synthetic quadratic datasets
- Polynomial regression using scikit-learn
- Linear, quadratic, and high-degree polynomial fitting
- Mean Squared Error (MSE) evaluation
- Understanding overfitting and underfitting behavior

### 2. Model Selection and Evaluation
- Receiver Operating Characteristic (ROC) curves
- Area Under the Curve (AUC)
- Confusion matrices
- Sensitivity and specificity
- F-Score calculation
- Comparative analysis of classification models
